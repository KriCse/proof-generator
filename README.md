# proof-generator
Proof generator using Haskell

## Propositional logic (0th order)
* *Variables*:  Should start with an upper case-letter (A, B, VarIaBle...)
* *Conjuction* (AND): ^
* *Disjunction* (OR): v (lower-case letter 'v')
* *Conditional* (IMPL, IF): -> 
* *Equality*: =
* *Inequalty* (XOR): !=
* *Negation* (NOT): !
